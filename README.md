# news_app
This project is a dedicated mobile application designed to deliver a fast, organized, and seamless news reading experience. The application focuses on aggregating and categorizing news content from various sources, providing regular updates, and featuring a modern, user-friendly interface that supports both light and dark themes.

1.Dynamic Content Categorization:
News articles are sorted into clear tabs for easy navigation.
The main categories displayed include "Business", "Sports", and "Science".
2.Efficient News Display:
News articles are presented in a structured list, with each item featuring a thumbnail image, the headline, and the publication date (using the ISO 8601 format, e.g., 2021-10-01T07:14:55Z).
Tapping an item leads to a detailed view, potentially opening the full article link within a web view.
3.Search and Interaction:Search Functionality:
A prominent search feature allows users to quickly find specific content across categories.
Settings: A clear settings icon is included for configuration options.
4.Flexible Design (UI/UX):The application supports both Dark Mode (Dark Theme) and Light Mode for optimal viewing comfort.
